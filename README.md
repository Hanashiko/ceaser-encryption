# Caesar Cipher (Ukrainian Alphabet)

A simple Caesar cipher implementation for the Ukrainian alphabet with a shift of 2.

## Usage

```bash
python main.py
```

Enter text in Ukrainian when prompted. The program outputs the encrypted result.

## How it works

Each letter is shifted by 2 positions in the Ukrainian alphabet (`–∞–±–≤–≥“С–і–µ—Ф–ґ–Ј–Є—Ц—Ч–є–Ї–ї–Љ–љ–Њ–њ—А—Б—В—Г—Д—Е—Ж—З—И—Й—М—О—П`). Non-alphabetic characters are preserved as-is.
